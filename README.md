# 内容分析器 Skill

用于分析保险行业资讯内容，提炼框架结构和写作风格，生成《保险行业资讯Top10》期刊。

## 主要功能

1. **内容获取**：支持PDF、Word、网页、微信公众号等多种输入
2. **联网搜索补全**：获取行业数据、政策背景、市场对比信息
3. **框架分析**：提炼框架结构和写作风格
4. **内容生成**：输出符合期刊风格的资讯内容
