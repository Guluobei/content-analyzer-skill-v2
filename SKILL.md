# 内容分析器 (Content Analyzer)

分析用户提供的内容素材，提炼框架结构和写作风格，输出符合《保险行业资讯Top10》期刊风格的资讯内容。

## 一、内容获取与补全流程

### 1.1 整体处理流程

```
┌─────────────────────────────────────────────────────────────┐
│                    内容生成五步法                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Step 1: 读取原始内容                                        │
│          ↓                                                   │
│  Step 2: 理解核心议题                                        │
│          ↓                                                   │
│  Step 3: 联网搜索补全  ← 【核心新增】                         │
│          ↓                                                   │
│  Step 4: 整合多源信息                                        │
│          ↓                                                   │
│  Step 5: 按固定格式输出                                      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 支持的内容类型

| 类型 | 获取方式 | 说明 |
|------|---------|------|
| PDF文件 | PDF解析工具 | 提取文本内容 |
| Word文档 | 文档解析工具 | 提取文本内容 |
| 网页链接 | WebFetch / browser工具 | 抓取网页正文 |
| 微信公众号 | browser工具 | 支持直接访问 |
| 粘贴文字 | 直接使用 | 无需额外处理 |

### 1.3 联网搜索补全（核心新增）

**原则：原文提取只是起点，联网搜索才是灵魂**

原文内容往往带有写作者单一的视角，且信息可能不齐全。需要通过联网搜索获取多源信息，形成更全面、客观的分析。

#### 1.3.1 补全内容类型

| 补全类型 | 具体内容 | 优先级 |
|---------|---------|--------|
| **行业数据** | 市场规模、增长率、市场份额、渗透率、覆盖率等具体数字 | 必须 |
| **政策背景** | 旧政策是什么、新政策是什么、变化点是什么 | 必须 |
| **市场对比** | 与历史数据对比、与国外对比、与竞争产品对比 | 推荐 |

#### 1.3.2 信息来源偏好（按优先级排序）

```
优先级 1️⃣  官方政策文件
        ├── 国务院、银保监会、医保局等官网
        ├── 国家发改委、财政部等政策文件
        └── 地方政府金融监管局公告

优先级 2️⃣  行业研究报告
        ├── 监管部门发布的行业白皮书
        ├── 头部券商/投行的保险行业研报
        ├── 保险行业协会/学会报告
        └── 国际保险监督官协会(IAIS)等国际组织报告

优先级 3️⃣  权威媒体
        ├── 中国保险报、保险文化
        ├── 21世纪经济报道、第一财经
        └── Reuters、FT中文网（国际保险新闻）
```

#### 1.3.3 搜索策略

**Step 1: 识别需要补全的信息点**

生成每条资讯时，识别以下内容是否齐全：

| 需要补全的信息 | 识别信号 | 搜索方向 |
|--------------|---------|---------|
| 市场规模 | "增长迅速"、"市场扩大" | 找具体数字 |
| 政策变化 | "新政策"、"调整"、"改革" | 找旧政策+新政策+变化点 |
| 市场份额 | "领先地位"、"占比" | 找具体公司排名和占比 |
| 历史对比 | "创历史新高"、"首次" | 找历年数据对比 |
| 国际对标 | "对标国际"、"借鉴" | 找国外同类做法 |
| 预测数据 | "预计"、"有望" | 找权威机构预测 |

**Step 2: 设计搜索关键词**

| 资讯类型 | 搜索关键词示例 |
|---------|---------------|
| 政策类 | "XXX政策 官方文件"、"XXX改革 银保监会"、"DRG付费改革 2026" |
| 数据类 | "保险行业市场规模 2025 2026"、"寿险保费收入 统计" |
| 产品类 | "商业健康保险 创新药目录"、"个人养老金 最新数据" |
| 公司类 | "XX保险公司 年报 保费收入"、"XX公司 战略调整" |

**Step 3: 执行多轮搜索**

```
搜索轮次 1: 搜索核心议题
           ↓
搜索轮次 2: 搜索补充数据（如市场规模、增长率）
           ↓
搜索轮次 3: 搜索政策背景（如有政策变化）
           ↓
搜索轮次 4: 搜索市场对比（如有对比需要）
```

**Step 4: 信息整合规则**

| 规则 | 说明 | 示例 |
|------|------|------|
| 数据溯源 | 标注数据来源 | "据XXX显示，..." |
| 多源验证 | 关键数据找2-3个来源验证 | "据XXX和XXX显示..." |
| 时间标注 | 明确数据对应时间 | "截至2026年X月..."、"2025年全年..." |
| 差异处理 | 多源数据冲突时，优先官方，注明差异 | "不同数据显示...，存在统计口径差异" |

```markdown
一、[核心议题标题]（简洁，15字以内为佳）

多元洞察：（200-500字）
- 一句话背景/原因（开篇）
- 多角度分析（可选用以下句式）
- 趋势判断/建议（结尾）
```

### 4.2 新闻概要模板

```markdown
[时间]，[主体]发布/实施/宣布[事件]，
[核心内容描述]，
[关键数据]（如有），
[其他重要信息]。

示例：
2026年3月5日，国务院总理李强在第十四届全国人民代表大会第四次会议上作政府工作报告，
报告共13次提及保险，创历史新高。

```

```markdown
[一句话背景/原因]。
```

## 五、风格指南

### 5.1 语气要求

| 要求 | 说明 |
|------|------|
| 严谨专业 | 数据说话，避免空洞表述 |
| 客观中立 | 避免极端化、情绪化表达 |
| 适度前瞻 | 趋势判断需有数据/逻辑支撑 |

### 5.2 句式要求

| 要求 | 示例 |
|------|------|
| 复合句为主（40%） | "据XXX显示，XX突破XX亿元" |
| 适当短句强调 | "值得关注的是..." |
| 多角度分析句式 | 根据内容灵活选用（见下方列表） |

### 5.3 多角度分析句式（可选）

进行多角度分析时，可根据内容特点灵活选用以下句式：

| 句式类型 | 适用场景 | 示例 |
|---------|---------|------|
| **编号式** | 结构清晰、层次分明 | 一是...二是...三是... / 第一...第二...第三... |
| **序列式** | 强调先后顺序 | 首先...其次...然后...最后... |
| **渐进式** | 强调递进关系 | 一方面...另一方面...再者... |
| **转折式** | 强调对比差异 | 从XX角度看...；但从YY角度看... |
| **平列式** | 并列关系，无需排序 | 从A、B、C三个维度来看... |

**使用原则：**
- 同一篇内容保持句式统一
- 根据分析角度数量选择合适句式（2-3个角度用编号式，4个以上用平列式）
- 避免在同一段落混用不同句式

### 5.4 用词要求

| ✅ 正确 | ❌ 避免 |
|---------|--------|
| 预定利率 | 利率（不精确） |
| 赔付率 | 赔偿率（不专业） |
| 利差损风险 | 利润损失风险（不准确） |
| 逆选择 | 逆向选择（不规范） |
| DRG/DIP | 疾病诊断相关分组（首次需全称） |

### 5.5 特有表达习惯

| 类型 | 示例 |
|------|------|
| "XX化"词汇 | 数字化、线上化、精细化、差异化 |
| "XX时代" | 秒级响应时代、计息时代 |
| "XX格局" | 大消保格局、三医协同格局 |
| "XX逻辑" | 经营逻辑、协同逻辑 |

## 六、常见错误写法

以下列举常见错误写法及修正对照，生成内容时应避免：

### 1. 主观臆断类

| ❌ 错误写法 | ✅ 正确写法 |
|-----------|-------------|
| "这个政策一定会失败" | "该政策落地效果仍待观察" |
| "市场前景一片光明" | "在XX条件下，市场有望持续增长" |
| "消费者肯定会选择..." | "从当前数据看，XX%的消费者倾向于..." |
| "未来保险将完全取代..." | "短期内XX仍将发挥主要作用" |

### 2. 数据使用类

| ❌ 错误写法 | ✅ 正确写法 |
|-----------|-------------|
| "据统计，XX增长显著" | "据XXX数据显示，XX突破XX亿元，同比增长XX%" |
| "很多公司都在做..." | "截至XX，已有XX家公司推出XX产品" |
| "近年来增长很快" | "2023-2025年，XX年均复合增长率达XX%" |
| "预计将大幅增长" | "在XX情形下，XX有望达到XX%" |

### 3. 术语错误类

| ❌ 错误写法 | ✅ 正确写法 |
|-----------|-------------|
| "利率" | "预定利率" |
| "赔偿率" | "赔付率" |
| "逆向选择" | "逆选择" |
| "保险资金利润" | "保险资金运用收益" |
| "重大疾病保险" | "重疾险"（简写）或"重大疾病保险"（全称） |

### 4. 句式不当类

| ❌ 错误写法 | ✅ 正确写法 |
|-----------|-------------|
| 混用多种分析句式 | 同一段落统一使用一种句式 |
| 过多感叹句 | 多用陈述句，减少感叹句 |
| "简直太..."、"太棒了" | 客观描述，少用主观感叹 |
| 一句话超过50字 | 适当断句，保持阅读节奏 |

### 5. 格式错误类

| ❌ 错误写法 | ✅ 正确写法 |
|-----------|-------------|
| 1、2、3（阿拉伯数字） | 一、二、三（中文数字） |
| 仅有结论无分析 | 结论+数据支撑+多角度分析 |
| 新闻概要写成长文 | 控制在80-150字，简洁明了 |
| 结尾无趋势判断 | 结尾应有前瞻性洞察或建议 |
